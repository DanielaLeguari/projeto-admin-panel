# Projeto Admin Panel

React com Context

Requisitos técnicos:
-  React + TypeScript;
-  React Hook Form;
-  toast (React Toastify, React Hot Toast, Sweet Alert2 ou outras);
-  loading (nprogress ou outras - também pode criar componente personalizado).

## Para iniciar o projeto você precisará dos comandos:

### `npm install`

### `npm start`
